Los Angeles 2019

Photos

Most pages use photos of Los Angeles and include some of my favorite locations. It's an ethnographic love story for LA. All the layouts will use CSS grid. I'll also bring in cool CSS properities like flexbox, shapes, transforms, animations and multi-column. Font selections come from Google and Adobe collections. I want to use variable fonts and web fonts served here. I'll keep a list of type specimens for review. If you'd like to view code or contact me find links in the footer.
